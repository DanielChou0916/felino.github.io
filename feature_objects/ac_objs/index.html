Allen-Cahn Damage Diffusion Programming Objects

In Felino, the action term in the phase-field equation is built on top of MOOSE's implementation. Below the kernels and action object used for the non-conserved phase-field diffusion are introduced.

\[ \dot d = -L\Bigl( g'(d)\,\psi^e + \frac{G_c\,f(\bar\alpha)}{c_0\,\ell}\,w'(d) - \nabla\!\cdot\!\Bigl(\frac{2\,G_c\,f(\bar\alpha)\,\ell}{c_0}\, \bar{\mathbf{A}}\cdot\nabla d\Bigr) \Bigr) \]

Three kernels are used to implement this diffusion:

\[ \underbrace{\dot d}_{\text{TimeDerivative/ADTimeDerivative}} +\; \underbrace{ L\,\Bigl( g'(d)\,\psi^e + \tfrac{G_c\,f(\bar\alpha)}{c_0\,\ell}\,w'(d) \Bigr) }_{\text{AllenCahn/ADAllenCahn}} \;-\; \underbrace{ L\,\Bigl( \nabla\!\cdot\!\bigl(\tfrac{2\,G_c\,f(\bar\alpha)\,\ell}{c_0}\, \bar{\mathbf{A}}\cdot\nabla d\bigr) \Bigr) }_{\text{ACInterfaceGradKappa/ADACInterfaceGradKappa}}=0 \]
  • TimeDerivative/ADTimeDerivative: computes the time derivative \(\dot d\).
  • AllenCahn/ADAllenCahn: handles the reaction term \(g'(d)\,\psi^e + \frac{G_c\,f(\bar\alpha)}{c_0\,\ell}w'(d)\).
  • ACInterfaceGradKappa/ADACInterfaceGradKappa: handles the diffusion term \(\nabla\cdot(\dots)\).

Programming object and example usages

PFNonconservedAction

This action object pipelines three kernels—ACInterfaceGradKappa, TimeDerivative, and AllenCahn—to solve the Allen–Cahn equation. The ACInterfaceGradKappa kernel is at the heart of Felino. PFNonconservedAction offers these options:

  • Set use_automatic_differentiation = true to enable automatic differentiation (AD).
  • Set use_grad_kappa = true to include spatial gradients of non-constant material parameters.
  • Handle gradient terms (\(\nabla\kappa\)) via AuxKernels and AuxVariables.
  • Auto-detect mesh dimension (e.g. requires grad_kappa_z in 3D).
  • Anisotropic PFF support is experimental. Set use_anisotropic_matrix = true and specify anisotropic_matrix = A_matrix.
  • If no anisotropic matrix is defined in the material block, the model defaults to isotropic PFF.
[Actions/PFNonconserved]
    [./d]
      free_energy                  = F
      kappa                        = kappa_op
      mobility                     = L
      variable_mobility            = false
      use_automatic_differentiation = true
      use_grad_kappa               = true
      grad_kappa_x                 = dkappa_dx
      grad_kappa_y                 = dkappa_dy
      use_anisotropic_matrix       = true
      anisotropic_matrix           = A_matrix
    [../]
  []

ACInterfaceGradKappa/ADACInterfaceGradKappa

This kernel handles the phase-field diffusion term (the Laplacian of \(d\)). It also includes:

  • Set use_grad_kappa = true for spatial gradients of non-constant parameters.
  • Handle \(\nabla\kappa\) via AuxKernels and AuxVariables.
  • Auto-detect mesh dimension (e.g. grad_kappa_z for 3D).
  • Anisotropic PFF: set use_anisotropic_matrix = true and anisotropic_matrix_name = A_matrix.
  • Without an anisotropic matrix in the material block, it falls back to isotropic PFF.
[Kernels]
    [./ac_interface_gradkappa]
      type                       = ADACInterfaceGradKappa
      variable                   = d
      mob_name                   = L
      kappa_name                 = kappa_op
      #args                      = T   # no need for constant L
      grad_kappa_x               = grad_kappa_x
      grad_kappa_y               = grad_kappa_y
      use_grad_kappa             = true
      variable_L                 = false
      use_anisotropic_matrix     = true
      anisotropic_matrix_name    = A_matrix
    [../]
  []
