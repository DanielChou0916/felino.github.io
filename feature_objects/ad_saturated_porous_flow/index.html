AD Saturated Poromechanics Programming Objects

In Felino, I added a small set of AD kernels to assemble saturated pore-pressure diffusion with automatic differentiation. The governing balance I target is:

\[ \rho_w\,\frac{\dot p_w}{M} \;+\;\rho_w\,\alpha_w\,\nabla\!\cdot\!\dot{\mathbf{u}} \;-\;\nabla\!\cdot\!\Big[\frac{\rho_w}{\mu_w}\,\mathbf{K}_w\cdot\big(\nabla p_w+\rho_w\mathbf{g}\big)\Big] \;=\;0 \quad \text{in } \Omega . \]

The element weak residual (ignoring the boundary integral from diffusion) is:

\[ \mathbf{r}_{p_w} = \int_\Omega \boldsymbol{N}_{p_w}\,\rho_w\,\frac{\dot p_w}{M}\,d\Omega \;+\;\int_\Omega \boldsymbol{N}_{p_w}\,\rho_w\,\alpha_w\,\nabla\!\cdot\!\dot{\mathbf{u}}\,d\Omega \;+\;\int_\Omega \boldsymbol{B}_{p_w}^\mathsf{T} \Big[\frac{\rho_w}{\mu_w}\,\mathbf{K}_w\cdot\big(\boldsymbol{B}_{p_w}\mathbf{p}_{w,e}+\rho_w\mathbf{g}\big)\Big]\,d\Omega . \]

These terms are split across four kernels:

\[ \underbrace{\int_\Omega\!\boldsymbol{N}_{p_w}\,\rho_w\,\tfrac{\dot p_w}{M}\,d\Omega}_{\text{ADHydroTimeDerivative}} +\underbrace{\int_\Omega\!\boldsymbol{N}_{p_w}\,\rho_w\,\alpha_w\,\nabla\!\cdot\!\dot{\mathbf{u}}\,d\Omega}_{\text{ADHydroVolStrainCoupling}} +\underbrace{\int_\Omega\!\boldsymbol{B}_{p_w}^\mathsf{T} \big[\tfrac{\rho_w}{\mu_w}\,\mathbf{K}_w\cdot \boldsymbol{B}_{p_w}\mathbf{p}_{w,e}\big]\,d\Omega}_{\text{ADHydroDiffusion}} +\underbrace{\int_\Omega\!\boldsymbol{B}_{p_w}^\mathsf{T} \big[\tfrac{\rho_w}{\mu_w}\,\mathbf{K}_w\cdot \rho_w\mathbf{g}\big]\,d\Omega}_{\text{ADHydroGravity}} \]
  • ADHydroTimeDerivative: time term \(\rho\,\dot p/M\). Uses AD so off-diagonal Jacobians are automatic.
  • ADHydroVolStrainCoupling: volumetric-strain source \(\rho\,\alpha\,\dot\varepsilon_v\). I provide \(\dot\varepsilon_v\) from a small AD material.
  • ADHydroDiffusion: symmetric diffusion \(\nabla N:\big(\frac{\rho}{\mu}\mathbf{K}\nabla p\big)\). Works with anisotropic \(\mathbf{K}\) (Rank-2 tensor).
  • ADHydroGravity: gravity head contribution \(\nabla N:\big(\frac{\rho}{\mu}\mathbf{K}\,\rho\mathbf{g}\big)\). Sign follows the usual convention (downward \(g_y<0\)).
  • ADMechanicsBiotPressure: couples \(p\) back into the solid via effective stress (not expanded here).

Programming object and example usages

ADHydroTimeDerivative / ADHydroVolStrainCoupling / ADHydroDiffusion

Minimal 2D poromechanics setup — pressure p plus solid displacements \((u_x,u_y)\). I keep names explicit for clarity.

  • variable = p selects pressure DOF and its test function; AD handles all cross-derivatives.
  • Time term needs fluid_density and biot_modulus (material props), units consistent (e.g. kg/m³, Pa, s).
  • Volumetric-strain source needs biot_coefficient and vol_strain_rate from a material (my ADHydroStrainRate).
  • Diffusion needs fluid_density, fluid_viscosity, and a tensor permeability_tensor (AD RankTwoTensor).
  • Gravity term is optional (same property set as diffusion). If you add it, pass your \(\mathbf{g}\) vector with your sign convention.
[Kernels]
  [poro_x]
    type = ADMechanicsBiotPressure
    p_eff_variable = p
    biot_coef_name = biot_coefficient
    variable = disp_x
    component = 0
  []
  [poro_y]
    type = ADMechanicsBiotPressure
    p_eff_variable = p
    biot_coef_name = biot_coefficient
    variable = disp_y
    component = 1
  []
  [TimeDerivative]
    type = ADHydroTimeDerivative
    variable = p
    fluid_density = fluid_density
    biot_modulus = biot_modulus
  []
  [Laplace_term]
    type = ADHydroDiffusion
    variable = p
    fluid_density = fluid_density
    fluid_viscosity = fluid_viscosity
    permeability_tensor = permeability_tensor
  []
  [vol_strain_coupling]
    type = ADHydroVolStrainCoupling
    variable = p
    fluid_density = fluid_density
    biot_coefficient = biot_coefficient
    vol_strain_rate = vol_strain_rate
  []
